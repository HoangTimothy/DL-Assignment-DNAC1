Introduction

This website is the landing page for three final assignments of the CO3133 course. Each project applies deep neural network architectures to a common data modality.

  • Image Data (Image Classification): Surveying Convolutional Neural Networks (CNN) and Vision Transformers (ViT).
  • Text Data (Text Classification): Surveying Recurrent Neural Networks (RNN) and Transformers.
  • Multimodal Data: Classification with zero-shot and few-shot learning methods.
